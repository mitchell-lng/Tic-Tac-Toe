# TicTacToe in CPP

## Description
Simple [Tic-Tac-Toe](https://en.wikipedia.org/wiki/Tic-tac-toe) Game made using C++

## Running the Game:
Compile:
```bash
g++ main.cpp
```

Windows Run:
```bash
start a.exe
```
Linux/Mac Run:
```bash
./a.out
```

## Game Settings
Values in inputs.txt can be changed in order to create different sized grids or different bot player modes
